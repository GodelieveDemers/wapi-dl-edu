  • phone_number: nomor WhatsApp.

  • access_token: token terenkripsi.
  • verify_token: token verifikasi webhook.

4. Konfigurasi ENV di VPS

Konfigurasi disimpan di file:

Contoh konfigurasi yang digunakan:

APP_URL=https://wapi.dl-edu.my.id
META_EMBEDDED_SIGNUP_APP_ID=750615218079136
META_EMBEDDED_SIGNUP_CONFIG_ID=1999076590997036
META_WA_API_VERSION=v25.0
META_WA_WEBHOOK_VERIFY_TOKEN=isi_token_random
META_WA_APP_SECRET=isi_app_secret_meta
META_EMBEDDED_SIGNUP_APP_SECRET=isi_app_secret_meta
Jangan tampilkan ke user: App Secret, Access Token, dan token credential lain. App ID dan Configuration ID boleh dipakai di frontend, tetapi lebih rapi disembunyikan dan hanya tampil sebagai status.

7. Embedded Signup

7.1 Fungsi

Embedded Signup memungkinkan customer menghubungkan WhatsApp Business Account miliknya melalui popup resmi Meta tanpa input manual WABA ID dan Phone Number ID.

7.2 Alur Teknis

User klik Hubungkan dengan Meta.
Aplikasi memanggil FB.login() dengan config_id.
Meta membuka popup onboarding.
Meta mengirim session info berisi WABA ID dan Phone Number ID.
Aplikasi mengirim response ke callback Laravel.
Laravel menyimpan channel ke whatsapp_cloud_channels.
Jika ada code dan App Secret valid, Laravel menukar code ke access token.

7.3 Batasan Penting

Meta membatasi Embedded Signup: onboarding customer melalui Embedded Signup hanya tersedia untuk BSP atau Tech Provider. Jika belum disetujui, Meta menampilkan pesan bahwa aplikasi tidak tersedia untuk Embedded Signup.

9. Review Meta, Business Verification, dan Tech Provider

9.1 Permission yang Diajukan

business_management
whatsapp_business_management
whatsapp_business_messaging

9.2 Contoh Alasan Pengajuan

Aplikasi WAPI dl-Edu digunakan sebagai platform SaaS untuk membantu pelanggan menghubungkan WhatsApp Business Cloud API mereka, mengelola channel, webhook, template pesan, status pesan, dan pengiriman pesan sesuai kebijakan WhatsApp Business Platform. Permission business_management, whatsapp_business_management, dan whatsapp_business_messaging diperlukan agar pelanggan dapat melakukan onboarding WhatsApp Business Account melalui Embedded Signup dan agar aplikasi dapat mengelola Phone Number ID, WABA ID, webhook, serta pengiriman pesan yang diminta pengguna.

9.3 Instruksi Review

1. Buka https://wapi.dl-edu.my.id/en/login
2. Login menggunakan akun demo reviewer.
3. Buka menu Devices.
4. Pilih tab WhatsApp Cloud API.
5. Klik Hubungkan dengan Meta.
6. Ikuti Embedded Signup untuk menghubungkan WhatsApp Business Account.
7. Setelah selesai, aplikasi menyimpan WABA ID dan Phone Number ID ke channel pengguna.

9.4 Verifikasi Bisnis

Jika perorangan, pilih Hak Milik Tunggal. Jika lembaga pendidikan, pilih Lembaga dan siapkan dokumen legal lembaga. Jika PT/CV, pilih jenis badan usaha sesuai dokumen.

  • Nama legal harus sesuai dokumen.
  • Alamat harus sesuai bukti alamat.
  • Nomor telepon dan email harus aktif.
  • Business portfolio harus terverifikasi untuk akses lanjutan.

  • Pending Business Verification Meta.
  • Pending Advanced Access permission WhatsApp.
